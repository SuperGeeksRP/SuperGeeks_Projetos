Star Wars

Desenvolvi uma API dedicada ao universo Star Wars, permitindo o acesso a diversos elementos

da franquia. O projeto oferece funcionalidades para recuperar informações sobre personagens,

planetas, naves e outros elementos do universo fictício. Utilizei tecnologias modernas para

garantir a eficiência e a facilidade de uso da API, proporcionando aos usuários uma experiência

completa e interativa ao explorar dados relacionados a Star Wars.

Calculadora

Desenvolvi uma calculadora funcional utilizando a linguagem JavaScript. A aplicação permite

realizar operações básicas, como adição, subtração, multiplicação e divisão, de forma intuitiva e

responsiva. Utilizei HTML e CSS para a estrutura e o design da interface, garantindo uma

experiência amigável ao usuário. A calculadora é acessível em navegadores web e foi projetada

para ser fácil de usar, proporcionando uma solução prática para cálculos simples.

Site do Batman

Desenvolvi um site dedicado ao Batman utilizando HTML e CSS. A plataforma destaca-se por

uma interface atrativa, incorporando elementos visuais e gráficos relacionados ao universo do

super-herói. Utilizei HTML para estruturar o conteúdo e CSS para estilizar, proporcionando uma

experiência visual envolvente. O site abrange informações sobre o Batman, sua

história e outros aspectos relevantes, oferecendo uma experiência completa para os fãs.
